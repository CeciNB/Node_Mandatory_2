## Remember to add .env

you might have to use `sudo `

```bash
touch .env
nano .env
```

add the following and fill out your info:

MAIL_HOST=
MAIL_PORT=
MAIL_USER=
MAIL_PASS=

or use .env.example for reference

## Installation

```bash
npm i
```

## Development

```bash
npm run dev
```
